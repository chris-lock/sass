Columns

Columns must always be wrapped inside a column-wrapper and will need both the column and column-width mixin if in a row with other columns. We use columns that have half the column gutter on the right and the left. This means we don't need a first or last class. Also, using box-sizing: border-box and fixed gutters allows us to easily nest columns. Adding a border directly to a column will make it too wide for the grid since there is already padding on there. Use the column-border mixin instead. 

							.example-columns {
								column-wrapper()
							}

							column-wrapper() must be included at every breakpoint to reset $column-margin.

						
									.example-column {
										column()
										column-width(3, 12)
										column-width(4, 12)
										column-width(6, 12)
									}
								
									column-width() must be included at every breakpoint to reset $column-gutter.
								
								The padding of the column-wrapper() plus the padding of the column-width() add up to equal the outer margin of the columns.
							
								The padding of the column-width() is half the column gutter so adjacent columns produce the gutter width.
							
								.offset {
									column-offset(1, 12, left)
									column-offset(1, 12, right)
								}
							
									.include-gutter {
										column-include-gutter()
									}
								
								.nested-1 {
									column()
									column-width(8, 12, true)
								}
							
									.nested-2 {
										column()
										column-width(4, 8)
									}
								
								.column-border-bad {
									border-top: 5px solid red
								}
							
								.column-border-good {
									column-border(top, 5px, green)
								}

								column-border() must be included at every breakpoint to reset the offset.
